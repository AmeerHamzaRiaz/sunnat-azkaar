# Nuxt 3 Starter Template

Starter Template for Nuxt 3

## Features

- [x] Vuetify
- [x] Eslint + Prettier
- [x] i18n
- [x] Pinia
- [ ] Auth 2.0
- [ ] Vitest


## Development Server

Start the development server on `http://localhost:3000`:

```bash
# pnpm
pnpm install
pnpm run dev
```

## Production

Build the application for production:

```bash
# pnpm
pnpm run build
```

Locally preview production build:

```bash
# pnpm
pnpm run preview
```
